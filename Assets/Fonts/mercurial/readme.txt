MERCURIAL is a free typeface based on contradiction. Alternating wavy and straight lines, it embodies the constant changes implied in the adjective meaning. MERCURIAL is free to download and use both for personal and commercial projects, however, sharing this project or just referencing this page would be greatly appreciated. Thank you.
